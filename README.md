A **full-life tracking dashboard** for daily, weekly, and monthly KPI tracking with interactive charts, streaks, and goal alerts. Designed for **personal productivity, habit tracking, and growth**.

## **Features**

### Daily Tracking
- Track **Faith**, **Family**, **Fitness**, and **Finance** pillars.
- Input numeric values or check completed tasks.
- Auto-calculate scores.

### Weekly & Monthly Insights
- Auto weekly average and monthly dashboard.
- 30-day challenge tracker.
- Weak pillar warnings.

- Auto goal percentage calculation.

### Export & Sync
- Export progress to PDF or JSON.
- Login-less cloud sync using LocalStorage.

### UI Features
- **Dark/Light mode toggle**
- Mobile & PC friendly (responsive)


## **How to Use**

1. Open `index.html` in your browser.
2. Enter your daily values for each pillar.
3. Check your weekly, monthly, and 30-day streaks.
4. Export progress whenever you want using PDF/JSON buttons.
5. Toggle **Dark/Light mode** for better visibility.
